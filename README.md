# GLAM Skill for Claude Code

A Claude Code skill for managing GLAM Protocol vaults and DeFi operations on Solana.

## What is GLAM Protocol?

GLAM provides programmable investment infrastructure on Solana through vaults with built-in access control, DeFi integrations, and tokenization capabilities.

## What does this skill do?

This skill enables Claude Code to help you:

- **Create and manage vaults** - Set up tokenized vaults with custom asset allowlists and access controls
- **Execute DeFi operations** - Swap tokens via Jupiter, lend/borrow on Kamino, trade perpetuals on Drift
- **Manage delegates** - Grant granular permissions to other wallets for vault operations
- **Handle investor flows** - Process subscriptions, redemptions, and share token management
- **Bridge assets** - Transfer USDC cross-chain via CCTP
